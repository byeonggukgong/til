# SQL 가져오기와 내보내기

## 가져오기

```bash
$ mysql -u 유저 이름 -p 데이터베이스 이름 < 파일 이름.sql
Enter Password: ****
```

## 내보내기

```bash
$ mysql -u 유저 이름 -p 데이터베이스 이름 > 파일 이름.sql
Enter Password: ****
```
